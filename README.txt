Panel Administrativo con React y Bootstrap

Este proyecto consiste en la creación de un panel administrativo web utilizando React con Vite como 
entorno de desarrollo. La interfaz fue diseñada con Bootstrap para lograr un estilo moderno y limpio.

El panel cuenta con un sistema de navegación compuesto por una barra superior Navbar con enlaces hacia las
secciones de Clientes, Proveedor, Usuarios y Logout. Cada sección está organizada como una vista 
independiente utilizando `react-router-dom` para el enrutamiento sin recargar la página.

Se implementó un diseño oscuro general, con texto claro y una fuente personalizada para mejorar la 
experiencia visual del usuario. Además, se aplicaron componentes estilizados como cards y botones 
utilizando clases de Bootstrap.

El proyecto fue desarrollado y estructurado en componentes independientes para facilitar su 
mantenimiento y escalabilidad.
